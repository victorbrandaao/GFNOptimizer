Transparent by design

No fake cleaner behavior.

CloudBoost does not install kernel extensions, hidden daemons, or permanent system patches. Some optimizations may request administrator permission because macOS protects network and system controls. Those changes are designed to be temporary and restored when CloudBoost is disabled or quits.

FAQ

Quick answers before installing.

Does CloudBoost work without PRO?

Yes. The free version includes GeForce NOW, xCloud, Local Game, manual boost, Balanced mode, and AWDL rollback protection.

Why does macOS ask for a password?

Some actions, such as temporary Wi-Fi interface tuning or process priority changes, require administrator approval on macOS.

Is it only for cloud gaming?

No. The 3.1 beta adds local Mac gaming profiles while keeping the existing cloud gaming features intact.

Where do I download it?

Use the latest DMG from the GitHub Releases page and drag CloudBoost.app to Applications.
