# CloudBoost 3.1.0 Customer Update

## English

Hi,

CloudBoost 3.1.0 is now available.

This is the stable release of the new Mac gaming build. The biggest change is that all core cloud and Mac gaming profiles are now free: GeForce NOW, Xbox Cloud Gaming, Boosteroid, Moonlight, VoidLink Extreme, Local Game, Steam, Epic Games, and Battle.net.

CloudBoost PRO now focuses on advanced add-ons: Auto-Detect, Auto Boost, Smart Boost, Stability Guard, Heat Guard, Keep Alive, diagnostics export, and advanced presets. Existing PRO customers keep access to every PRO feature they already purchased.

What changed:

- Cloud and Mac gaming profiles are now free.
- The UI is cleaner and more compact.
- Missing platform apps no longer block Boost activation.
- Network path detection is more reliable.
- The direct updater still downloads the latest DMG from GitHub releases.

Download:
https://github.com/victorbrandaao/CloudBoost/releases/latest

If macOS shows an "App is damaged" message after installing, run:

```bash
xattr -cr /Applications/"CloudBoost.app"
```

Thanks for supporting CloudBoost. This update is meant to make the free version more useful while keeping PRO focused on automation and power-user features.
